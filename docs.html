Building from Source

The xkbEdit Linux Keyboard Interpreter is written in C++. To build it from source:

  1. Ensure cmake and g++ are installed.
  2. Clone the repo: git clone https://github.com/idempomlieko/xkbEdit.git
  3. Build: cd xkbEdit && mkdir build && cd build && cmake .. && make

Dependencies

  • X11 libraries
  • xkbcommon (for keyboard layouts)
  • Optional: GTK or Qt for the UI

Configuration

Layouts and preferences are stored in ~/.config/linux-keyboard-interpreter/config.json.
